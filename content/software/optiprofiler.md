[OptiProfiler](https://optiprofiler.com) is a benchmarking tool for optimization solvers.
The current version only supports derivative-free optimization solvers.

It is currently implemented in [Python](https://www.python.org) and [MATLAB](https://www.mathworks.com/products/matlab.html).

- The [documentation](https://optiprofiler.com) is under construction.
- The source files can be downloaded from [GitHub](https://github.com/OptiProfiler/OptiProfiler).
- Compiled versions for Python will be available from PyPI soon.
