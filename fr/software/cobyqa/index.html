COBYQA

2022-10-21
1 min de lecture
Optimization
DFO

COBYQA (Constrained Optimization BY Quadratic Approximations) est un solveur d’optimisation sans dérivées pour résoudre des problèmes d’optimisation contraints, en utilisant des approximations quadratiques.

Il est actuellement implémenté en Python, mais une version Fortran du software (et des interfaces pour MATLAB et Python) apparaîtra prochainement.

  • Une documentation complète est accessible sur https://www.cobyqa.com/.
  • Les fichiers sources peuvent être téléchargés depuis https://github.com/ragonneau/cobyqa/.
  • Des versions compilées peuvent être installées depuis https://pypi.org/project/cobyqa/.
  • Une description détaillée de l’algorithme et de son implémentation est donnée dans ma thèses de doctorat.
