PDFO

2022-10-21
1 min de lecture
Optimization
DFO

PDFO (Powell’s Derivative-Free Optimization solvers) est une librairie multiplateforme MATLAB et Python fournissant des interfaces pour l’utilisation des solveurs sans dérivées de M. J. D. Powell. Elle comprend UOBYQA, NEWUOA, BOBYQA, LINCOA et COBYLA.

  • Une documentation complète est disponible.
  • Les fichiers sources peuvent être téléchargés depuis GitHub.
  • Des versions compilées pour Python peuvent être installées depuis PyPI.
